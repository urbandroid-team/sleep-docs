    Sleep mask

      • Sleep masks types
        • How to configure cable type Sleep Mask
        • How to configure Bluetooth type Sleep Mask V2

    Sleep mask makes your sleep light stimulated. It can help you fall sleep faster, wake up better and help you with snoring or lucid dreaming.

    Dedicated to be used with Sleep as Android and Mindroid.

    • Learn more on a dedicated website.
    • Get it from Happy Electronics.
    • Sunrise alarm: from low red to full yellow, no more annoying alarm sounds
    • Lucid dreaming: learn to control and remember your dreams
    • Anti-snoring
    • Comfortable to wear all night
    • Light stimulated lullabies for faster fall asleep (CAUTION: Wired version only, Wireless version only supports lullabies through Mindroid app with Sleep program)
    • More light stimulation programs with Mindroid app

    Settings -> Services -> Smartlight -> Use Sleep Mask (both types) Settings -> Services -> Smartlight -> Connect Sleep Mask V2 (Only BT Sleep Mask V2)

    Sleep masks types#

    .Cable type Sleep mask

    .Sleep mask V2 (wireless, connects via BT)

    How to configure cable type Sleep Mask#

    1. In Sleep as Android, go to Settings -> Services -> Smartlight -> Enable (at the very top) to enable smartlight functions in the app.
    2. Connect the small black box to the black connector via USB.
    3. Connect the cable to the micro USB port of your Android device. A diode lights on.
    4. In the Sleep as Android Settings -> Services -> Smartlight ,scroll down to the Sleep Mask section, tick the Use SleepMask.

    Note: All smartlight features are enabled automatically now - including anti-snoring alerts, lucid dreaming cues, and sunrise alarm.

    How to configure Bluetooth type Sleep Mask V2#

    1. Power on the Sleep Mask using an ON/OFF switch.
    2. On your mobile device, make sure that Bluetooth is enabled and the Sleep app has Location permissiom (required for BT scanning).
    3. In Sleep as Android, go to Settings -> Services -> Smartlight -> Enable (at the very top) to enable smartlight functions in the app.
    4. Then, scroll down to the Sleep Mask section, tick the Use SleepMask.
    5. Tap on Connect Sleep Mask V2.
    6. Wait till the Sleep Mask is listed in the dialogue and select it.
    7. Now the mask is paired (you will see a BT address below Connect Sleep Mask V2). The Mask also blinks to confirm the connection.

    Note: All smartlight features are enabled automatically now - including anti-snoring alerts, lucid dreaming cues, and sunrise alarm. Note: The app cannot connect BT mask and Sleep Phaser at the same time.
