    Sleep as Android does not have direct integration with Mi Bands and Amazfit, because the API necessary for integration is not open. But thanks to the Notify app, you can use some devices with Sleep app. Notify for Mi Band, Notify for Xiaomi & Mi Fitness and Notify for Amazfit & Zepp are developed by OneZerobit.

    You can visit Notify app page for Sleep as Android integration here.

    a|.Enable Sleep as Android integration.

    |===

    How to set up integration in Sleep app#

    1. Open Settings -> Sleep tracking -> Wearables -> Use wearable.
    2. Choose the correct option from the list.
    3. For HR data enable Settings -> Sleep tracking -> Wearable -> Heart rate monitoring.

    Note: Sleep tracking collects data the whole night and they are analyzed in real-time. This means the watch must stay connected with BT during sleep tracking.

    HR monitoring#

    1. Enable collecting HR data by the Sleep app in N&F -> Sleep section -> Sleep as Android -> Settings -> Heart monitor
    2. Enable collecting dat in Sleep app in Settings -> Sleep tracking -> Wearables -> Heart rate monitoring.

    [cols="^,^"] |===

    a|.Open Settings for Sleep as Android integration.

    a|.Notify&Fintess HR data enabled for Sleep

    |===

    Button snooze / dismiss#

    [cols="^,^"] |===

    a|.Open Settings for Sleep as Android integration.

    a|.Choose the action of the button.

    |===

    • Dismiss - enables dismissing alarm with the button (dismiss will be confirmed with a vibration)
    • Snooze - with two button press, you can snooze the alarm (will be confirmed with a vibration)
    • No action for the button - disable both previous options

    Custom vibration pattern#

    1. First, enable wearable vibrations in Sleep app in Settings -> Sleep tracking -> Wearables -> Alarm.
    2. Open Notify app and enable custom vibrations in Sleep as Android settings section.
    3. Open the vibration settings with the Vibration button.
    4. Adjust the vibration pattern - you can change intensity and duration, and also vibration delay.
    5. Test the resulting vibration pattern with the Play button in the top right corner.

    Note: You can also have a silent alarm with vibrations only, for details, see the guide)

    [cols="^,^"] |===

    a|.Open Settings for Sleep as Android integration.

    a|.Find Custom vibration.

    |===
